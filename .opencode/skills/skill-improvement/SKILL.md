# Skill Improvement Skill

## Process

1. Read `.agent/known-failures.md`, `.agent/memory.md`, and recent scorecards.
2. Find repeated failure patterns.
3. Write a concrete improvement proposal in `.agent/skill-backlog.md`.
4. Do not make large SKILL.md changes without review.
5. Prefer specific next-step rules over broad warnings.
